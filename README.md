# 🌱 環保替代品查詢表

## 專案簡介
環保替代品查詢表是一個幫助消費者快速找到環保替代商品的互動式網頁工具。使用者可以透過搜尋關鍵字或分類篩選（廚房、浴室、外出、清潔、女性用品、3C周邊），查詢日常生活中常見商品的環保替代方案，包括替代品名稱、價格範圍、優缺點、環保評分以及使用小提示。

## 專案目標（SDGs）
本專案對應 **聯合國永續發展目標 SDG 12：責任消費與生產**，旨在：
- 提供透明且易於查詢的環保替代商品資訊
- 降低消費者購買環保產品的選擇障礙
- 引導大眾轉向更永續的消費模式

## 功能特色
- 🔍 **關鍵字搜尋**：輸入商品名稱即可快速查詢
- 🏷️ **分類篩選**：6大分類（廚房、浴室、外出、清潔、女性用品、3C周邊）
- 💰 **真實價格**：價格參考台灣各大電商平台（MOMO、PChome、博客來、東森購物、樂天市場）
- ⭐ **環保評分**：每項替代品皆有1~5星評分
- 💡 **實用小提示**：每項商品附帶實用的使用建議
- 🖼️ **圖片支援**：可自行放入商品圖片

## 如何執行
1. 將所有檔案下載到同一資料夾中
2. 在 `images` 資料夾中放入圖片（命名為 1.png ~ 30.png）
3. 用瀏覽器打開 `index.html` 即可使用
4. 建議使用 **VS Code Live Server** 開啟以獲得最佳體驗

## 資料來源
- 價格參考：MOMO購物網、PChome 24h購物、博客來、東森購物、樂天市場（2025-2026年市售價格）
- 產品資訊：環保署綠色生活網、環保團體建議、市售環保產品彙整
